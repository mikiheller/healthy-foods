## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file with your OpenAI API key:
```bash
OPENAI_API_KEY=your-api-key-here
```

3. Generate food images:
```bash
npm run generate-images
```

4. Run the development server:
```bash
npm run dev
```
